# 枪手
## 角色详情

求生者{
    角色名称：猎手
    生命值上限：24
    初始生命值：24
    潜行值：9（饥饿状态下为8）
    天赋[侦察]:{
        行动条件：所在地块有相邻的且未被展示的地图块。
        行动效果：最多展示两个相邻的地图块，且不触发任何地块触发效果。
    }
}

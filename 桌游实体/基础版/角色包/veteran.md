# 枪手
## 角色详情

求生者{
    角色名称：老兵
    生命值上限：22
    初始生命值：22
    潜行值：7（饥饿状态下为6）
    天赋[把你的爪子拿开]:{
        行动：对老兵造成2点伤害，然后狗直到你的下回合开始免疫伤害。 
    }
}

求生者{
    角色名称：狗
    生命值上限：12
    初始生命值：12
    潜行值：9（饥饿状态下为8）
    天赋[咬他们]:{
        射程：中距离
        行动：对目标造成3点伤害，然后狗受到一点伤害。
    }
}

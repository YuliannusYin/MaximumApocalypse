# 机械师
## 角色详情

求生者{
    角色名称：机械师
    生命值上限：26
    初始生命值：26
    潜行值：8（饥饿状态下为7）
    天赋[维修]:{
        行动条件：场上的所有弃牌堆中至少有1张装备牌
        行动效果：从任一弃牌堆中抽一张装备牌，并把它放置在场上任一玩家的装备区中。
    }
}

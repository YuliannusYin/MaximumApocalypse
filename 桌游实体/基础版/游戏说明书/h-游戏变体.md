# 难度变体
当你发现在进行几次游戏之后，缺乏挑战性那么你可以使用下面几点来使游戏变得更难
加入更多伏击或一无所获卡牌到拾荒牌库。移除3个怪物标记。
玩家随机投掷骰子来决定初始饥饿等级。
向地图中加入更多地图块。

# 单人变体
如果你单人进行本游戏，我们建议你选择两名不同的角色来进行。把这两个牌库一起洗混然后抓8张卡牌作为初始手牌。你的手牌上限是20。你可以打出手中的任何手牌到任一角色上。除此之外，你就要按照通常规则继续进行游戏,不断在两名角色间切换,轮流进行回合执行行动。
这是一种更简便的模拟多角色的单人模式。

# 末日变体
如果你追求挑战，你可以自由的混合怪物牌库和/或加入额外的地图块进行游戏。这会让游戏更难预测且更困难。你能在核弹爆炸、外星人入侵之后的末日启示录中生存下来么?

# 老兵和狗注意事项
当扮演老兵和狗进行游戏时，你可以选择哪个敌人攻击哪个角色，但是必须尽可能平均的在角色间分配敌人。例如，如果你面前有两个敌人一个会攻击老兵，另一个会攻击狗，但是你可以选择哪个攻击老兵哪个攻击狗。如果一名敌人攻击所有射程内的玩家，那么它既攻击老兵，又攻击狗。
老兵的存储容量上限是4，狗的存储容量是0。如果两个角色都没有死亡，那么使用两个角色中最低的潜行数值进行潜行检定。如果一名角色死亡，你可以继续进行游戏，直到另一名角色也死亡。

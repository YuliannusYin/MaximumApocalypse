## 地图块详情

格式如下：
    地图块{
        地图块名称：
        触发：
        效果：
    }

地图块{
    地图块名称：百货商店
    展示：执行一次免费的拾荒行动
}

地图块{
    地图块名称：避难所
    触发：当你本回合开始时不在这里且在这里结束回合时
    效果：你免疫伤害
}

地图块{
    地图块名称：城市街道
    进入：抓一张怪物卡
}

地图块{
    地图块名称：电厂
    进入：弃掉所有的食物卡，中毒层数加一。
}

地图块{
    地图块名称：河流
    触发：进入此地块时，进行一次潜行检定
    效果：成功：移动到本地图块；失败返回之前的地图块
}

地图块{
    地图块名称：机场
    行动：如果这里没有怪物，移动到另一个已展示的地图块。
}

地图块{
    地图块名称：警察局
    展示：执行一次免费的拾荒行动
}

地图块{
    地图块名称：军事基地
    进入：对你面前的所有怪物各造成两点伤害
}

地图块{
    地图块名称：监狱
    展示：立即结束你的回合
    进入：减少一次行动次数
}

地图块{
    地图块名称：旷野
    展示：抓一张怪物卡
    进入：抓一张怪物卡
}

地图块{
    地图块名称：工厂
    展示：向所有相邻的地块各增加一个怪物标记
}

地图块{
    地图块名称：购物中心
    展示：执行一次免费的拾荒行动
}

地图块{
    地图块名称：加油站
    展示：执行一次免费的拾荒行动
}

地图块{
    地图块名称：绿洲
    结束：减少一点饥饿值
}

地图块{
    地图块名称：面包车 #特殊地图块一般是玩家的出生点和结束点
}

地图块{
    地图块名称：墓地
    展示：每名玩家抓一张怪物卡
    进入：抓一张怪物卡
}

地图块{
    地图块名称：农场
    展示：执行一次免费的拾荒行动
}

地图块{
    地图块名称：强盗营地
    进入：弃掉一张已装备的装备卡或者受到5点伤害
}

地图块{
    地图块名称：山
    进入：抓一张牌
}

地图块{
    地图块名称：隧道
    行动：移动到另一个已经展示的【隧道】地图块
}

地图块{
    地图块名称：森林
    触发：进入此地块后，并在同一个回合内移动出此地块
    效果：抓一张怪物卡
}

地图块{
    地图块名称：沙漠
    进入：饥饿等级增加1
}

地图块{
    地图块名称：游乐园
    展示：弃掉三张牌
    触发：当你本回合结束时，在此地块上
    效果：弃掉一张牌
}

地图块{
    地图块名称：医院
    进入：恢复一点生命值
    触发：当你本回合结束时，在此地块上
    效果：恢复两点生命值
}

地图块{
    地图块名称：坠毁点
    展示：所有玩家销毁一张装备
    进入：销毁一张牌
}

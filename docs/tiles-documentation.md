## 地块概述

**总地块数量**: 31种

**地块定义**:
地块是游戏地图的基本单元，每个地块包含以下属性：
- **地块名称**: 地块的唯一标识
- **拾荒标记**: 可拾荒的颜色（无、蓝、绿、红）
- **怪物出生值**: 触发怪物出生的骰子值
- **触发效果**: 特定条件下触发的效果

## 特殊机制说明

### 1. 拾荒标记机制

#### 普通拾荒标记
- **蓝色**: 抓蓝色拾荒牌库的卡牌
- **绿色**: 抓绿色拾荒牌库的卡牌
- **红色**: 抓红色拾荒牌库的卡牌

#### 数值拾荒标记（城市街道）
- **蓝5**: 骰子值为5时可拾荒蓝色
- **红6**: 骰子值为6时可拾荒红色
- **绿8**: 骰子值为8时可拾荒绿色

**拾荒规则**:
- 拾荒需要消耗1个行动点
- 拾荒后触发地块的"结束"效果
- 某些地块提供免费拾荒机会

---

### 2. 怪物出生值机制

#### 怪物出生流程
1. 每回合开始时投掷两颗骰子（2-12）
2. 检查所有已展示的地块
3. 在匹配怪物出生值的地块上放置怪物标记
4. 每个地块最多放置3个怪物标记
5. 怪物标记池总数30个

#### 特殊出生值
- **0**: 军事基地永远不会出生怪物
- **2, 12**: 避难所只在极端骰子值时出生怪物
- **多值**: 某些地块有多个出生值，增加出生概率

---

### 3. 触发效果时机

#### 展示效果
- **触发时机**: 地块首次被探索并展示时
- **触发次数**: 每个地块只触发一次
- **示例**: 工厂向相邻地块增加怪物标记

#### 进入效果
- **触发时机**: 玩家移动进入地块时
- **触发次数**: 每次进入都触发
- **示例**: 医院恢复生命值

#### 结束效果
- **触发时机**: 玩家在地块上执行拾荒或结束回合时
- **触发次数**: 每次结束都触发
- **示例**: 花园治愈状态效果

#### 行动效果
- **触发时机**: 玩家在地块上消耗行动点时
- **触发次数**: 需要主动触发
- **示例**: 机场移动到其他已展示地块

#### 停止效果
- **触发时机**: 玩家停止移动并选择继续移动时
- **触发次数**: 特定条件下触发
- **示例**: 森林抓怪物卡

#### 检定效果
- **触发时机**: 需要进行潜行检定或其他检定
- **触发次数**: 根据检定结果触发
- **示例**: 河流移动检定

---

### 4. 特殊地块效果

#### 免费拾荒
- **地块**: 百货商店、加油站、警察局、农场、购物中心
- **效果**: 展示时执行一次免费拾荒行动
- **价值**: 节省1个行动点

#### 恢复生命
- **地块**: 医院
- **效果**: 进入恢复1点，结束恢复2点
- **策略**: 在医院结束回合可获得更多恢复

#### 减少饥饿
- **地块**: 绿洲
- **效果**: 结束时饥饿等级减少1点
- **价值**: 缓解饥饿压力

#### 怪物控制
- **军事基地**: 进入时对所有怪物造成伤害
- **教堂**: 进入时击晕怪物
- **城堡**: 进入时所有怪物回满血（负面效果）

#### 快速移动
- **机场**: 移动到其他已展示地块
- **隧道**: 移动到其他隧道地块
- **河流**: 检定成功才能移动

#### 免疫伤害
- **避难所**: 特定条件下免疫伤害
- **策略**: 在避难所结束回合可获得保护

---

### 5. 负面效果地块

#### 高风险地块
- **坠骇点**: 摧毁装备、烧毁卡牌
- **电厂**: 弃掉食物卡、中毒
- **城堡**: 所有怪物回满血
- **监狱**: 结束回合、失去行动点
- **强盗营地**: 弃掉装备或受伤害

#### 怪物密集地块
- **荒野**: 展示和结束都抓怪物卡
- **墓地**: 每名玩家抓怪物卡
- **祭坛**: 放置3个怪物标记
- **工厂**: 向相邻地块增加怪物标记

#### 饥饿压力地块
- **沙漠**: 进入时饥饿等级增加1
- **绿洲**: 结束时饥饿等级减少1（正面）

## 地块数据结构

### TileData 类定义

```gdscript
class_name TileData extends Resource

# 地块名称
@export var tile_name: String = "未知地块"

# 地块描述
@export var description: String = ""

# 拾荒颜色（无、蓝、绿、红）
@export var scavange_color: String = "无"

# 拾荒颜色数值要求（城市街道特殊机制）
@export var scavange_values: Dictionary = {}

# 怪物出生值列表
@export var monster_spawn_values: Array[int] = []

# 触发条件列表（展示、进入、结束、行动、停止、检定）
@export var trigger_conditions: Array[String] = []

# 效果描述列表
@export var effect_descriptions: Array[String] = []

# 地块图标（用文字代替）
@export var icon_text: String = "地块"

# 是否已展示
var is_explored: bool = false

# 当前怪物标记数量
var monster_markers_count: int = 0
```

### 数据文件示例

```
[gd_resource type="TileData" load_steps=2 format=3 uid="uid://tile_hospital"]

[ext_resource type="Script" path="res://godot/data/tile_data.gd" id="1"]

[resource]
script = ExtResource("1")
tile_name = "医院"
description = "提供医疗服务的场所"
scavange_color = "红"
scavange_values = {}
monster_spawn_values = [3]
trigger_conditions = ["进入", "结束"]
effect_descriptions = ["恢复1点生命", "恢复2点生命"]
icon_text = "医院"
```

## 地块策略建议

### 优先探索地块
1. **医院**: 恢复生命，红色拾荒
2. **警察局**: 蓝色拾荒，免费拾荒
3. **农场**: 绿色拾荒，免费拾荒
4. **加油站**: 红色拾荒，免费拾荒，燃料
5. **军事基地**: 怪物伤害，无怪物出生

### 避免探索地块
1. **城堡**: 怪物回满血
2. **祭坛**: 3个怪物标记
3. **监狱**: 结束回合，失去行动点
4. **坠骇点**: 摧毁装备
5. **电厂**: 中毒效果

### 策略性使用地块
1. **避难所**: 在这里结束回合免疫伤害
2. **绿洲**: 结束回合减少饥饿等级
3. **隧道**: 快速移动到其他隧道
4. **机场**: 快速移动到已展示地块
5. **花园**: 结束回合治愈状态效果

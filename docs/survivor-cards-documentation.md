## 角色天赋详解

### 枪手 - 快速拔枪
- **触发时机1**：游戏开始时
- **触发效果1**：将牌库中的装备牌【柯尔特手枪】装备你的装备区
- **触发时机2**：当你受到饥饿伤害时
- **触发效果2**：将装备区或弃牌区中的【柯尔特手枪】重新洗回你的牌库

### 消防员 - 拳打
- **射程**：短距离
- **行动条件**：无
- **行动效果**：对一个目标造成2点伤害

### 猎手 - 侦察
- **行动条件**：所在地块有相邻的且未被展示的地图块
- **行动效果**：最多展示两个相邻的地图块，且不触发任何地块触发效果

### 机械师 - 维修
- **行动条件**：场上的所有弃牌堆中至少有1张装备牌
- **行动效果**：从任一弃牌堆中抽一张装备牌，并把它放置在场上任一玩家的装备区中

### 外科医生 - 缝合
- **射程**：短距离
- **行动效果**：使一个目标回复一点生命

### 老兵 - 把你的爪子拿开
- **行动效果**：对老兵造成2点伤害，然后狗直到你的下回合开始免疫伤害

### 狗 - 咬他们
- **射程**：中距离
- **行动效果**：对目标造成3点伤害，然后狗受到一点伤害

## 数据结构定义

### PlayerData.gd 属性映射

```gdscript
class_name PlayerData
extends Resource

# 基础属性
@export var id: String                    # 角色唯一标识符
@export var character_name: String        # 角色名称
@export var max_hp: int                   # 生命值上限
@export var current_hp: int               # 当前生命值

# 饥饿系统
@export var hunger_level: int             # 饥饿等级
@export var is_starving: bool             # 是否饥饿状态
@export var starving_damage_stage: int    # 饥饿伤害阶段

# 位置与属性
@export var position: Vector2i            # 地图位置
@export var base_stealth: int             # 正常潜行值
@export var starving_stealth: int         # 饥饿状态潜行值
@export var action_points: int            # 行动点数

# 状态效果
@export var poison_tokens: int            # 中毒标记数
@export var is_stunned: bool              # 是否被击晕

# 局内动态数据
var hand: Array[CardRuntime]              # 手牌
var equipment_zone: Array[CardRuntime]    # 装备区
var deck: Array[CardRuntime]              # 牌库
var discard_pile: Array[CardRuntime]      # 弃牌堆
```

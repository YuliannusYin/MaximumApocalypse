## 拾荒卡概述

**拾荒卡定义**:
拾荒卡是玩家在地图上进行拾荒行动时获得的卡牌，分为不同颜色类别，每种颜色对应不同的资源类型。

**拾荒卡颜色分类**:
- **红色**: 燃料、解毒剂、医疗用品
- **绿色**: 食物
- **蓝色**: 防弹背心、对讲机、手枪、背包、手电筒、双筒望远镜、弹药、炸药
- **灰色**: 一无所获、伏击、多余配件、脏毯子、旧报纸、满是灰尘的笔记本

### GDScript 数据类定义

```gdscript
# [新增] 2026-06-24: 拾荒卡数据类，用于存储拾荒卡的基本属性和效果配置
class_name ScavengeCardData
extends Resource

# [新增] 2026-06-24: 拾荒卡唯一标识符
@export var id: String = ""

# [新增] 2026-06-24: 拾荒卡名称
@export var card_name: String = ""

# [新增] 2026-06-24: 拾荒卡大类（战备、食物、补给、特殊）
@export var category: String = ""

# [新增] 2026-06-24: 拾荒卡颜色（红色、绿色、蓝色、灰色）
@export var color: Enums.ScavengeColor = Enums.ScavengeColor.NONE

# [新增] 2026-06-24: 拾荒卡类型（行动牌、装备牌）
@export var card_type: String = ""

# [新增] 2026-06-24: 拾荒卡数值（弹药数量、饥饿值减少量等）
@export var value: int = 0

# [新增] 2026-06-24: 拾荒卡效果描述
@export_multiline var effect: String = ""

# [新增] 2026-06-24: 拾荒卡效果脚本ID，用于EffectManager匹配执行具体效果
@export var effect_script_id: String = ""
```

## 拾荒卡使用规则

### 1. 拾荒行动触发
- **触发条件**: 玩家在具有拾荒标记的地块上执行拾荒行动
- **拾荒颜色**: 地块的拾荒标记颜色决定可抓取的拾荒卡颜色
- **行动消耗**: 拾荒行动消耗1个行动点

### 2. 拾荒卡抓取
- **抓取数量**: 每次拾荒行动抓取1张对应颜色的拾荒卡
- **牌库管理**: 每种颜色的拾荒卡有独立的牌库和弃牌堆
- **牌库洗混**: 牌库空时，将弃牌堆洗混后重新使用

### 3. 拾荒卡使用
- **行动牌**: 可立即使用或保存到手中
- **装备牌**: 需要装备到装备区才能使用
- **使用时机**: 根据卡牌类型决定使用时机

### 4. 特殊拾荒卡
- **灰色卡牌**: 可能带来负面效果，需要谨慎处理
- **数值范围**: 某些卡牌的数值可能是范围值（如X ∈ {1,2}）

## 拾荒卡策略建议

### 优先拾荒地块
- **医院**: 红色拾荒，可获得燃料、解毒剂、医疗用品
- **农场**: 绿色拾荒，可获得食物，减少饥饿值
- **警察局**: 蓝色拾荒，可获得弹药，补充武器

### 拾荒时机选择
- **饥饿值高时**: 优先寻找绿色拾荒地块
- **武器弹药不足时**: 优先寻找蓝色拾荒地块
- **需要特殊补给时**: 优先寻找红色拾荒地块

### 灰色卡牌处理
- **谨慎拾荒**: 灰色地块可能带来负面效果
- **风险评估**: 根据当前状态决定是否拾荒

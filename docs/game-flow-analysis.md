# 末日启示录桌游全流程解析

**整理日期**: 2026-06-24
**版本**: 基础版
**来源**: 桌游实体/基础版文档

---

## 一、游戏概述

### 1.1 游戏简介
末日启示录（Maximum Apocalypse）是一款合作型卡牌桌游，每名玩家扮演后启示录时代的求生者，通过卡牌执行行动和获取装备，在怪物威胁下探索地区、填饱肚子并完成任务目标，最终成功逃脱。

### 1.2 游戏配件
- **核心组件**: 拾荒板、角色卡（7张）、立像（6个）、塑料底座（6个）
- **卡牌**: 求生者卡（180张）、怪物卡（100张）、首领卡（8张）、任务卡（13张）、拾荒卡（80张）、参考卡（6张）
- **标记**: 地图块（38个）、卡档（12个）、弹药标记（30个）、燃料标记（12个）、怪物标记（30个）、生命标记（57个）、中毒标记（20个）、目标标记（3个）
- **骰子**: 大骰子（2个）、小骰子（6个）

### 1.3 游戏目标
每个任务设定不同的剧本和目标。大多数任务要求玩家完成目标并带着一定数量的燃料返回面包车来逃离区域。

## 二、游戏设置流程

### 2.1 基础设置步骤

**步骤1: 选择角色**
- 每名玩家选择一个求生者
- 将角色卡、立像和牌库放到自己的游戏区域

**步骤2: 选择任务**
- 玩家可以选择特定任务或随机选择
- 阅读任务中的额外设置说明

**步骤3: 构建牌库**
- 拿取对应的怪物卡并洗混组成怪物牌库
- 根据任务说明构建三个拾荒牌库（蓝、绿、红）
- 分别混洗这些牌库

**步骤4: 构建地图**
- 把初始地图块（面包车）放在一边
- 把任务中包含的地图块洗混为牌库
- 将地图块正面向下放置成网格状
- 在最后的地图块摆放后，把"面包车"地图块放到与地图最外围的任何一个地图块相连的位置
- 所有玩家在该位置开始游戏（除非任务有特殊说明）

**步骤5: 初始手牌**
- 每名玩家从求生者牌库中抓4张牌作为初始手牌
- 手牌上限是10张
- 每名玩家可以进行一次重调：把最多4张刚抓的卡牌放到一边，抓取等量的卡牌，再把放到一边的卡牌洗回牌库

**步骤6: 饥饿设置**
- 每名玩家拿取一个小骰子放到角色卡上
- 把骰子点数设置为"1"，代表初始饥饿等级

**步骤7: 初始怪物**
- 每名玩家抓一张怪物卡放在自己面前
- 怪物卡会停留在面前（即使移动了）直到被消灭或赶走
- 玩家可以攻击其他玩家面前的怪物（只要在攻击范围内）

**步骤8: 确定先手**
- 选择一名先手玩家
- 按照顺时针顺序依次进行游戏

### 2.2 游戏区域布局

**公共区域**:
- A. 怪物弃牌堆
- B. 怪物牌库（洗混并放在便于够到的位置）
- C. 拾荒牌库（洗混并放在便于抓取的位置）
- J. 地图

**玩家区域**:
- D. 拾荒弃牌堆
- E. 求生者弃牌堆
- F. 求生者牌库
- G. 角色卡
- H. 初始怪物
- I. 初始地点

## 三、游戏回合流程

### 3.1 进入玩家回合

**阶段1: 怪物出生**
- 投掷两颗大骰子，记两个骰子点数之和为X
- 在每个正面朝上、且与X点数相同的地图块上放置一个怪物标记
- 每个地图块的怪物标记上限是3
- 不能在已经有3个怪物标记的地图块上再添加标记
- 如果玩家所在地块已经有3个标记且与X点数相同，则直接抓一张怪物卡放到自己面前
- 所有在该地图块上的玩家都必须抓一张怪物卡

**阶段2: 抽牌**
- 从求生者牌库抓取一张牌
- 当求生者牌库空了不能抓牌时，玩家被消灭

**阶段3: 执行行动（4个行动点）**

**可执行的行动**（可任意组合，可重复执行）:
1. 横向或竖向移动1格
2. 从求生者牌库中抓1张牌
3. 从手牌中打出1张牌
4. 执行1张已经在游戏中的卡牌上的行动
5. 拾荒：根据当前地点（如果可以的话）抓1张拾荒卡

**免费行动**（不消耗行动点）:
- 每回合一次：弃掉两张求生者卡牌来从牌库抓一张新卡牌
- 每回合一次：和另一名与你在同一地点的玩家交易拾荒卡
- 根据目标牌的指示打出卡牌（如把燃料卡打出加进面包车）

**阶段4: 增加饥饿值**
- 把饥饿骰子翻转到对应的面来增加饥饿等级1点
- 如果已经处于饥饿状态，增加饥饿伤害等级
- 如果饥饿等级到6，把角色卡牌翻面并受到饥饿伤害
- 饥饿伤害不可被减少或避免
- 在接下来的回合，如果没能饱餐一顿，会受到越来越多的伤害

**阶段5: 怪物攻击**
- 在回合结束时，面前的所有怪物都会攻击并造成伤害
- 游戏按照顺时针进行

## 四、核心机制解析

### 4.1 饥饿状态机制

**饥饿等级 progression**:
- 正常状态：饥饿等级1-5
- 饥饿状态：饥饿等级6及以上

**进入饥饿状态**:
- 当饥饿等级到达6，开始承受饥饿伤害
- 把角色卡翻面，饥饿骰放在饥饿记录栏的顶端（数字2）
- 不能再使用自己的天赋行动

**饥饿伤害 progression**:
- 饥饿等级6 → 承受2点伤害 → 饥饿骰子移到数字4
- 饥饿等级继续增加 → 承受4点伤害 → 饥饿骰子移到数字6
- 饥饿等级继续增加 → 承受6点伤害 → 饥饿骰子移到数字8
- 饥饿等级继续增加 → 承受8点伤害 → 饥饿骰子移到"死亡"
- 饥饿等级继续增加 → 死亡

**解除饥饿状态**:
- 吃了一些东西或者把饥饿等级减到6以下时，把角色卡翻面
- 如果又变为饥饿状态，角色卡再翻面，饥饿骰子放到饥饿记录栏的顶端重新开始

### 4.2 移动机制

**基本规则**:
- 可以移动到任何相邻的地图块
- 不能斜向移动
- 尝试移动到一个地图块需要花费1个行动
- 移动时展示地图块并按照文字效果执行

**进入有怪物标记的地图块**:
- 如果面前没有任何怪物，必须进行一次潜行检定来悄悄溜走
- 如果想要抓怪物卡，可以决定跳过检定并移除所有怪物标记，然后每移除一个标记就抓一张怪物卡

### 4.3 潜行检定机制

**检定流程**:
- 投两个主骰子
- 如果两个骰子的和不大于求生者的潜行数值，检定成功
- 地图块上每有一个怪物标记，潜行数值就会减1
- 如果两个骰子的和大于潜行数值，检定失败：移除该地图块上的所有怪物标记，每移除一个怪物标记就抓一张怪物卡

**特殊情况**:
- 如果怪物牌库抓空了，重洗怪物弃牌堆并组成新怪物牌库

### 4.4 打出卡牌机制

**卡牌类型**:
- 即时行动卡：作为一个行动打出，按照卡牌文字执行，然后弃掉
- 装备卡：装备到装备区，持续生效

**打出规则**:
- 从手牌中打出一张卡牌需要花费一个行动
- 装填武器、吃食物和治疗玩家都需要花费行动
- 在打出一张拾荒卡后，把它放到拾荒弃牌堆中，不要放到求生者弃牌堆中

**交易规则**:
- 作为免费行动，可以向与你处于同一地点的其它玩家给出/拿取/交易拾荒卡
- 不能交易求生者卡

## 五、地块系统解析

### 5.1 地块属性

**地块名称**: 地块的名字（如"城市街道"）

**拾荒标记**: 代表在本地块进行"拾荒"行动时，可以从哪个颜色的拾荒牌堆中抓牌
- 如果地块左上角没有任何标记，就不能在本地图块进行"拾荒"

**怪物出生点数**: 在"怪物出生"流程中，怪物会在本地图块上出现

**地块事件触发条件**: 触发效果的状态
- 展示：当该地图块第一次被翻开时触发
- 进入：当你每次进入这个地图块时触发
- 结束：当你回合结束时，你在该地点，就会触发
- 行动：花费行动触发
- 停止：结束你本回合的移动或触发效果

**地图块效果**: 几乎每个地点都有效果

### 5.2 特殊地块示例

**面包车**:
- 拾荒标记：无
- 怪物出生值：6
- 触发效果：无
- 作用：起始点和终点，任务完成时需要返回这里

**医院**:
- 拾荒标记：红
- 怪物出生值：3
- 触发效果：
  - 进入：恢复1点生命
  - 结束：恢复2点生命

**警察局**:
- 拾荒标记：蓝
- 怪物出生值：6
- 触发效果：
  - 展示：执行一次免费的拾荒行动

**加油站**:
- 拾荒标记：红
- 怪物出生值：4、5、9
- 触发效果：
  - 展示：执行一次免费的拾荒行动

## 六、怪物系统解析

### 6.1 怪物类别

**基础版包含4个怪物包**:
1. 外星人类怪包
2. 突变体怪包
3. 机器人类怪包
4. 僵尸类怪包

### 6.2 怪物属性

**怪物名**: 怪物的名称

**怪物级别**:
- 首领：最强大的怪物，通常在任务特定时刻出现
- 精英：较强的怪物
- 普通：基础怪物

**初始血量**: 怪物的生命值上限

**攻击伤害**: 怪物攻击时造成的伤害

**射程**:
- 无：只能攻击自己所纠缠的玩家
- 短距离：可以攻击自己所纠缠玩家所在格子的所有玩家
- 中距离：可以攻击自己所纠缠的玩家所在格子和前后左右相邻格子的所有玩家
- 长距离：可以攻击自己所纠缠的玩家和相邻的玩家和更远的所有玩家

**效果**: 怪物的特殊能力

**怪物类别**: 怪物所属的类型

### 6.3 怪物示例

**僵尸女王（首领）**:
- 初始血量：18
- 攻击伤害：5
- 射程：无
- 天赋：当场上任何僵尸类型怪物被消灭时，消灭该怪物的玩家必须抓取一张新的怪物卡

**外星收割者（首领）**:
- 初始血量：25
- 攻击伤害：5
- 射程：无
- 天赋1：该怪物卡被玩家抓取时，摧毁场上所有的装备牌
- 天赋2：该怪物卡攻击时，随机烧毁目标玩家区域内的一张牌

## 七、角色系统解析

### 7.1 角色属性

**角色名**: 求生者的名称

**初始生命**: 角色的生命值上限

**潜行数值**: 用于潜行检定的数值，饥饿状态下会降低

### 7.2 角色示例

**枪手**:

  - 受到饥饿伤害时：将装备区或弃牌区中的【柯尔特手枪】重新洗回牌库

- 行动射程

## 八、攻击射程解析

### 8.1 怪物射程

**无射程**:
- 只能攻击自己所纠缠的玩家

**短距离**:
- 可以攻击自己所纠缠玩家所在格子的所有玩家
- 例如：玩家在格子（0,0），怪物可以攻击这个格子上的所有玩家

**中距离**:
- 可以攻击自己所纠缠的玩家所在格子和前后左右相邻格子的所有玩家
- 例如：玩家在格子（0,0），怪物可以攻击格子（0,0）、（1,0）、（-1,0）、（0,1）、（0,-1）上的所有玩家

**长距离**:
- 可以攻击自己所纠缠的玩家和相邻的玩家和更远的所有玩家
- 例如：玩家在格子（0,0），怪物可以攻击格子（0,0）、（-2~2,0）、（0,-2~2）、（1,1）、（-1,-1）、（1,-1）、（-1,1）上的所有玩家

### 8.2 玩家射程

**短距离**:
- 可以在角色当前地点距离内指定目标或产生效果

**中距离**:
- 可以在角色当前地点和横竖相邻的地点内指定目标或产生效果

**长距离**:
- 可以在距角色当前地点不超过2格横竖相邻的地点内指定目标或产生效果，但是不包含当前地点
- 例如：玩家在格子（0,0）上，可以指定格子（-2~2,0）、（0,-2~2）、（1,1）、（-1,-1）、（1,-1）、（-1,1）上的目标或产生效果，注意没有（0,0）

## 九、游戏结束条件

### 9.1 失败条件

**玩家被消灭**:
- 没有剩余生命
- 牌库中没有卡牌（不能抓牌）

**被消灭后的处理**:
- 把角色面前的所有怪物弃掉，替换为怪物标记
- 该角色携带的所有拾荒卡视为在他死亡的那个地点（可以放在该地图块上，之后可以被其它玩家捡起）

**团队失败**:
- 所有玩家被消灭
- 当要放置怪物标记时，已经没有可用的怪物标记了

**增加难度规则**:
- 可以使用"当任何玩家被消灭时，你们输掉游戏"来代替上述规则

### 9.2 胜利条件

**基本胜利**:
- 完成了任务并且收集到了所有需要的燃料（作为一个团队）
- 所有人都返回到了面包车上

**胜利触发时机**:
- 在玩家的回合结束时，胜利条件才触发
- 玩家依然会在他们回合结束前受到伤害

## 十、游戏术语表

**行动**: 需要花费一个行动来触发该卡牌上的文字效果

**攻击**: 每次该怪物攻击时触发该效果

**烧毁**: 弃掉牌库顶牌

**摧毁**: 当该怪物被消灭时触发该效果

**弃牌**: 弃掉本牌来触发本牌上的文字效果。弃牌不算做是一个行动

**结束**: 当你回合结束时，你在该地点，那么就会触发该效果

**进入**: 当你进入该地点时触发该效果

**展示**: 当该地图块第一次被翻开时触发该效果

**被动**: 只要该牌在游戏中，该牌就会一直起效果

**中毒**: 把一个中毒标记放到角色卡上。在每回合结束时，受到等同于角色卡上的中毒标记数目的伤害。这是一个状态效果

**停止**: 结束你本回合的移动或触发效果

**击晕**: 目标直到你的下回合开始都不会造成伤害

**检定**: 进行一次潜行检定

## 十一、示例回合流程

### 11.1 完整回合示例

**角色**: 枪手
**状态**: 面前有一只僵尸步行者，装备柯尔特手枪，饥饿等级4，枪有4发子弹，角色和消防员在加油站(9)

**回合流程**:

**1. 怪物出生**:
- 投骰子，投出了6
- 把各一个怪物标记放到展示的警察局(6)和面包车上

**2. 抽牌**:
- 抓了一张牌

**3. 执行行动（4个行动点）**:

**第一个行动**:
- 使用枪的行动，花费1弹药对僵尸步行者造成2点伤害
- 僵尸步行者还有2点生命值

**第二个行动**:
- 在加油站进行拾荒，从红色牌库中抓了一张牌；是一张燃料
- 根据该卡的效果，要么立即装备它作为免费行动，要么弃掉它
- 决定装备它，此时还剩余2个存储空间

**免费行动**:
- 从与自己在同一位置的消防员那里拿了一张食物卡

**第三个行动**:
- 移动到相邻的地图块并展示了它。是城市街区
- 因为城市街区拥有"进入：抓一张怪物卡"的效果，所以抓了一张怪物卡，是一张僵尸喷吐者

**第四个行动**:
- 花费了另一个弹药来再次攻击僵尸步行者
- 它被杀死，放到怪物弃牌堆中

**4. 增加饥饿值**:
- 把饥饿等级增加到5

**5. 怪物攻击**:
- 僵尸喷吐者进行了攻击，对枪手造成2点伤害
- 相邻地点的消防员也受到2点伤害，因为僵尸喷吐者会攻击中距离的所有角色

**回合结束**

## 十二、策略建议

### 12.1 团队协作策略

**资源共享**:
- 及时交易拾荒卡，特别是食物和燃料
- 在同一地点时充分利用免费交易

**分工合作**:
- 根据角色特长分配任务
- 例如枪手负责战斗，消防员负责救援

**位置协调**:
- 避免分散太远，保持相互支援的距离
- 注意怪物的射程范围

### 12.2 饥饿管理策略

**预防饥饿**:
- 优先收集食物卡
- 合理规划行动，避免不必要的饥饿增加

**应对饥饿**:
- 及时使用食物降低饥饿等级
- 在饥饿状态下优先解决饥饿问题

### 12.3 怪物应对策略

**怪物标记管理**:
- 注意怪物出生点数，避免聚集在高风险地块
- 及时清理怪物标记，避免达到上限

**怪物战斗**:
- 优先消灭高威胁怪物
- 注意怪物的特殊效果和射程

**潜行策略**:
- 在潜行值高时优先尝试潜行检定
- 在怪物标记多时考虑直接抓怪物卡

### 12.4 地块利用策略

**地块选择**:
- 优先探索有拾荒标记的地块
- 注意地块的触发效果

**地块效果利用**:
- 利用免费拾荒行动的地块
- 利用治疗和恢复效果的地块

**地块风险规避**:
- 避免负面效果的地块
- 注意地块的怪物出生值

## 十三、常见问题解答

### 13.1 游戏设置问题

**Q: 地图块如何摆放？**
A: 把任务中包含的地图块洗混为牌库，正面向下放置成网格状。在最后的地图块摆放后，把"面包车"地图块放到与地图最外围的任何一个地图块相连的位置。

**Q: 初始手牌如何重调？**
A: 每名玩家可以进行一次重调，把最多4张刚抓的卡牌放到一边，抓取等量的卡牌，再把放到一边的卡牌洗回牌库。

### 13.2 回合流程问题

**Q: 怪物出生时如何处理？**
A: 投掷两颗大骰子，记两个骰子点数之和为X。在每个正面朝上、且与X点数相同的地图块上放置一个怪物标记（上限3）。如果玩家所在地块已经有3个标记且与X点数相同，则直接抓一张怪物卡。

**Q: 行动点如何分配？**
A: 每回合有4个行动点，可以任意组合执行移动、抓牌、打出卡牌、执行卡牌行动、拾荒等行动。可以多次执行同一行动。

### 13.3 饥饿状态问题

**Q: 饥饿伤害如何计算？**
A: 当饥饿等级到达6，开始承受饥饿伤害。饥饿伤害 progression：2点、4点、6点、8点、死亡。饥饿伤害不可被减少或避免。

**Q: 如何解除饥饿状态？**
A: 吃了一些东西或者把饥饿等级减到6以下时，把角色卡翻面，恢复正常状态。

### 13.4 怪物问题

**Q: 怪物射程如何理解？**
A: 无射程只能攻击自己所纠缠的玩家；短距离攻击同一格子的所有玩家；中距离攻击相邻格子；长距离攻击更远的格子。

**Q: 如何进行潜行检定？**
A: 投两个主骰子，如果两个骰子的和不大于求生者的潜行数值，检定成功。地图块上每有一个怪物标记，潜行数值就会减1。

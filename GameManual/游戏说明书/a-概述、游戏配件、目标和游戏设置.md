# 概述

末日启示录（Maximum Apocalypse）是一款每名玩家扮演后启示录求生者，用卡牌来执行行动和获取装备的合作游戏。玩家需要在不断面对怪物的同时，探索各个地区、填饱肚子并找寻他们完成任务、成功逃脱所需要的资源。
最后，玩家要么一同完成目标并逃离启示录的废土，要么就一起死在这里。

# 目标

每个任务都为求生者设定了不同的剧本和目标。
大多数任务会要求你们完成目标并带着一定数目的燃料返回到面包车上来逃离该区域。

# 游戏设置

1. 每名玩家选择一个求生者，把该求生者的角色卡、立像和牌库放到自己的游戏区域。
2. 玩家可以选择要进行的任务，或者随机选择一个。阅读该任务中的额外设置说明。
3. 拿取对应的怪物卡并把他们洗混组成怪物牌库。
4. 根据任务说明构建三个不同的拾荒牌库（蓝、绿、红）。分别混洗这些牌库
5. 构建地图:
   把初始地图块(面包车)放在一边并把该任务中包含的地图块洗混为一个牌库。然后把地图块正面向下放置成一个网格状，代表本任务的地图。你可以摆成简单的方形或者按照喜好摆放。在最后的地图块摆放后，把"面包车"地图块放到与地图最外围的任何一个地图块相连的位置。所有玩家在该位置开始游戏(除非任务中有特殊说明)。
6.在地图构建完成后,每名玩家从对应的求生者牌库中抓4张牌作为初始手牌。
   每名玩家的手牌上限是10。
   每名玩家可以进行一次重调，把最多4张刚抓的卡牌放到一边，让后抓取等量的卡牌，再把放到一边的卡牌洗回牌库。
7.每名玩家同时拿取一个小骰子放到他们的角色卡上。把骰子点数设置为"1"，代表每名玩家的初始饥饿等级。
8.每名玩家抓一张怪物卡放在自己的角色面前(你们太吵了，很多怪物被吸引了过来)。
   怪物卡会停留在你的面前(即使你移动了)直到他们被消灭或通过其它卡牌的效果被赶走。
   玩家可以攻击其它玩家面前的怪物，只要该怪物在该玩家的攻击范围内。
9.在完成这些设置后，选择一名先手玩家，然后按照顺时针的顺序依次进行游戏。
使用游戏中的各种标记来记录伤害、弹药等。你也可以使用骰子、笔和纸等其它记录工具来记录这些状态。

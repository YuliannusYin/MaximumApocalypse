# 饥饿状态

当你的饥饿等级到达6，你就开始承受饥饿伤害。
把你的角色卡翻面并把饥饿骰放在饥记录栏的顶端。
当你处于饥饿状态时，你不能再使用自己的天赋行动。
每回合当你需要再增加饥饿等级时，把它沿着饥饿记录栏向卡（2/4/6/8/死亡）进行并承受对应的伤害。
当你吃了一些东西或者把饥饿等级减到6以下时，把你的角色卡翻面。
如果你又变为饥饿状态,把你的角色卡再翻面同时把饥饿骰子放到饥饿记录栏的顶端。
比如：玩家回合内饥饿值加到6点，角色牌翻面，饥饿骰子放到饥饿记录栏的顶端数字2。
    下个回合内饥饿值若要继续增加，则改为扣两点生命值，然后将饥饿骰子下移到饥饿记录栏的数字4的位置。
    下个回合内饥饿值若要继续增加，则改为扣四点生命值，然后将饥饿骰子下移到饥饿记录栏的数字6的位置。
    下个回合内饥饿值若要继续增加，则改为扣六点生命值，然后将饥饿骰子下移到饥饿记录栏的数字8的位置。
    下个回合内饥饿值若要继续增加，则改为扣八点生命值，然后将饥饿骰子下移到饥饿记录栏的死亡的位置。
    下个回合内饥饿值若要继续增加，则改为死亡。
    若玩家在饥饿状态下，将饥饿值降低到6以下，角色牌翻回正面。
    若后面又重新进入饥饿状态，角色牌翻面，饥饿骰子放到饥饿记录栏的顶端重新开始记录。


# 移动

玩家可以移动到任何相邻的地图块但是不能斜向移动。尝试移动到一个地图块需要花费1个行动。当你试图移动到一个地图块时，展示它并按照文字效果执行(如果能够执行的话)。

列如，你进入并展示了一个强盗营地地图块，触发了它的效果。你必须决定要么承受伤害要么弃掉一张装备着的装备卡。

如果你进入了一个有怪物标记的地图块并且面前没有任何怪物，你必须进行一次潜行检定来悄悄溜走。如果你想要抓怪物卡,你可以决定跳过这个检定并移除所有怪物标记，然后每移除一个标记就抓一张怪物卡。

# 潜行检定

投两个主骰子。如果两个骰子的和不大于你求生者的潜行数值，那么你就成功了。地图块上每有一个怪物标记，你的潜行数值就会减1。如果两个骰子的和大于你的潜行数值，那么移除该地图块上的所有怪物标记,每移除一个怪物标记就抓一张怪物卡。
如果怪物牌库抓空了，重洗怪物弃牌堆并组成一个新怪物牌库。

# 打出卡牌

在你的回合能打出的卡牌一共有两种:即时行动和装备。拾荒卡和求生者卡都属于其中的某一种。从手牌中打出一张卡牌需要花费一个行动。即时卡牌作为一个行动打出;按照卡牌文字执行，然后弃掉。
装填武器、吃食物和治疗玩家都需要花费行动。在打出一张拾荒卡后，把它放到拾荒弃牌堆中，不要放到你的弃牌堆中。

注意:
作为一个免费行动，你可以向与你处于同一地点的其它玩家给出/拿取/交易拾荒卡。你不能交易你的求生者卡。

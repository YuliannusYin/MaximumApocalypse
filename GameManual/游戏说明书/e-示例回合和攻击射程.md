# 示例回合

我扮演枪手进行游戏，且面前有一只僵尸步行者，我装备着我的柯尔特手枪。
我的饥饿等级目前是4，但是我的枪还有4发子弹。我的角色和消防员在加油站(9)。
我投骰子来进行怪物出生，投出了6。
我把各一个怪物标记放到展示的警察局(6)和面包车上。
我抓了一张牌并执行我的4个行动:
第一个行动:我使用我枪的行动并花费1弹药来对僵尸步行者造成2点伤害。
它还有2点生命值。第二个行动:我在加油站进行拾荒并从红色牌库中抓了一张牌;是一张燃料。
根据该卡的效果，我要么立即装备它作为免费行动，要么弃掉它。
我决定装备它，此时我还剩余2个存储空间。免费行动:我从与我在同一位置的消防员那里拿了一张食物卡。
第三个行动:我移动到相邻的地图块并展示了它。是城市街区。
因为城市街区拥有"进入:抓一张怪物卡"的效果，所以我抓了一张怪物卡，是一张僵尸喷吐者。
第四个行动:我花费了另一个弹药来再次攻击僵尸步行者。
它被杀死,放到怪物弃牌堆中。
我把我的饥饿等级增加到5。然后我面前的僵尸喷吐者进行了攻击，对我造成2点伤害。
相邻地点的消防员也受到2点伤害，因为僵尸喷吐者会攻击中距离的所有角色。我的回合结束了。

# 攻击射程

|(x,y)|x=-2|-1|0|1|2|
|---|---|---|---|---|---|
|y=-2|(-2,-2)|(-1,-2)|(0,-2)|(1,-2)|(2,-2)|
|-1|(-2,-1)|(-1,-1)|(0,-1)|(1,-1)|(2,-1)|
|0|(-2,0)|(-1,0)|(0,0)|(1,0)|(2,0)|
|1|(-2,1)|(-1,1)|(0,1)|(1,1)|(2,1)|
|2|(-2,2)|(-1,2)|(0,2)|(1,2)|(2,2)|

怪物射程{ 
    无：无射程的怪物只能攻击自己所纠缠的玩家
    短距离：怪物可以攻击自己所纠缠玩家所在格子的所有玩家.
        比如玩家在格子（0，0），怪物可以攻击这个格子上的所有玩家。
    中距离：怪物可以攻击自己所纠缠的玩所在格子和前后左右相邻格子的所有玩家。
        比如玩家在格子（0，0），怪物可以攻击格子（0，0）、（1，0）、（-1，0）、（0，1）、（0，-1）上的所有玩家。
    长距离：怪物可以攻击自己所纠缠的玩家和相邻的玩家和更远的所有玩家。
        比如玩家在格子（0，0），怪物可以攻击格子（0，0）、（-2~2，0）、（0，-2~2）、（1，1）、（-1，-1）、（1，-1）、（-1，1）上的所有玩家。
}

玩家射程{
    短距离:这些卡牌可以在你角色当前地点距离内指定目标或产生效果。
    中距离:这些卡牌可以在你角色当前地点和横竖相邻的地点内指定目标或产生效果。
    长距离:这些卡牌可以在距你角色当前地点不超过2格横竖相邻的地点内指定目标或产生效果，但是不包含你当前地点。
        比如玩家在格子（0，0）上，玩家可以指定格子（-2~2，0）、（0，-2~2）、（1，1）、（-1，-1）、（1，-1）、（-1，1）上的目标或产生效果，注意没有（0，0）。
}

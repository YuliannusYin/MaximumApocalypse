# 末日启示录 PC 版

> 基于实体桌游《Maximum Apocalypse》的 PC 单机重制版，使用 Godot 4.7 + GDScript 开发。

## 项目简介

《末日启示录 PC 版》是一款 1-6 人合作类后启示录求生游戏，将原版桌游《Maximum Apocalypse》的核心体验移植到 PC 平台。

文明已经崩溃，玩家扮演在僵尸、突变体、机器人、外星人四类末日威胁中挣扎求生的幸存者，需要探索随机生成的地图、收集物资、击杀怪物，最终完成指定任务目标并驾驶面包车逃离。

## 玩法概述

- **合作求生**：1-6 名玩家共同完成任务，任意一名幸存者死亡都会加重团队压力，全员死亡即失败
- **卡牌驱动**：每名幸存者拥有专属牌库，武器、道具、技能均以卡牌形式呈现
- **回合制流程**：每回合依次经历「怪物出生 → 抓牌 → 行动 → 回合结束结算」四个阶段，每回合 4 个行动点
- **程序化地图**：任务卡决定地块布局，地块随机填充，每局地图皆不同，重玩性强
- **四类末日威胁**：僵尸、突变体、机器人、外星人，各自拥有独立的怪物牌库与行为模式
- **拾荒系统**：红 / 绿 / 蓝三色拾荒牌堆按地块颜色抽取，分别偏向燃料药品、食物、弹药
- **饥饿机制**：忽略进食会逐步加重饥饿，最终造成不可减免的伤害
- **12 个任务**：每个任务有不同的胜利条件、地图布局、首领配置与特殊规则

## 当前状态

本项目处于**早期开发阶段**，目前尚无可玩版本发布。

主菜单与基础 UI 框架已搭建，核心游戏循环（GameSession、TurnManager、Skill 系统等）仍在设计与实现中。请关注后续更新。
